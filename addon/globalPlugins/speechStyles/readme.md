Speech Styles add-on scaffold
=============================

This package contains a lightweight implementation of the requested Speech Styles add-on behavior for NVDA.

It supports:
- multiple built-in speech styles such as Windows 10 Narrator, Windows XP, Windows 7 Narrator, and JAWS
- element-level enable/disable toggles using checkboxes
- pause choices of none, comma, and period
- announcement placement before, after, or none
- JSON-backed persistence for custom configuration

In a real NVDA development environment, this can be extended to hook into the full speech framework and install as a proper add-on.
